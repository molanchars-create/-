## 论文摘要

本文构建了一个微观基础化的新经济地理学(NEG)模型，整合了三大机制：
1. **垂直关联**：上游电池/零部件供应商与下游整车厂之间的投入产出联系
2. **地方政府补贴**：降低企业有效固定成本
3. **Marshall知识外部性**：产业集群中的本地化技术溢出

模型生成5个可检验假说(H1-H5)，使用2019-2024年297个中国地级市面板数据和空间杜宾模型(SDM)进行实证检验。

### 核心发现

| 假说 | 理论预测 | 实证结果 | 一致性 |
|------|---------|---------|--------|
| H1 前向关联 | β₁>0 | β₁=2.15*** | ✓ |
| H2 补贴主效应 | β₂>0 | β₂=19.04*** | ✓ |
| H3 替代效应 | β₃<0 | β₃≈0 或 β₃>0 | ✗ 方向相反 |
| H4 空间异质 | 东西差异 | 待检验 | — |
| H5 迟滞效应 | 路径依赖 | 待检验 | — |

**关键意外发现**：H3符号反转——补贴与产业链在现实中是**互补品**而非替代品，反映出中国地方政府"顺势而为"的政策特征。
